# warga-management-system
Web-based application for managing neighborhood administration, resident data, and community operations Built with Laravel &amp; MySQL for real-world usage

📌 Overview
This system is designed to help manage administrative and operational activities in residential communities such as:
* Resident data management
* Monthly dues (iuran) tracking
* Announcement & communication
* Financial reporting
* Community service records
Built to simplify manual processes into a centralized digital system.

🎯 Purpose
* Digitize neighborhood administration (RT/RW / perumahan)
* Improve transparency & reporting
* Simplify communication between admins and residents
* Reduce manual bookkeeping

⚙️ Tech Stack
* Laravel – Backend Framework
* MySQL – Database
* Blade Template – Frontend
* Bootstrap – UI

  
✨ Key Features :

👨‍👩‍👧 Resident Management
* Add / edit / delete resident data
* Family grouping (Kartu Keluarga)
* Address & contact tracking

💰 Iuran & Financial System
* Monthly contribution tracking
* Payment status (paid / unpaid)
* Financial reports
* Cash flow monitoring

📢 Announcement System
* Broadcast information to residents
* Event notifications
* Important alerts

📊 Dashboard & Reporting
* Summary of residents
* Payment statistics
* Monthly reports
* Export to PDF / Excel

🔐 User Role Management
* Admin (pengurus)
* Staff
* Resident (warga)
